### 2. 实施阶段

#### 翻译层增强（第二层优化）

**文件**：`utils/translator.py`

**新增功能**：
1. **情绪标记系统**（9种情绪）
   - [平静地] - 正常叙述
   - [兴奋地] - 激动、惊喜
   - [强调地] - 重点强调
   - [思考地] - 思考、犹豫
   - [疑问地] - 提问、好奇
   - [解释地] - 专业解释
   - [总结地] - 归纳、收尾
   - [快速地] - 列举数据
   - [缓慢地] - 强调每个字

2. **重音标记系统**
   - 用【】标记关键词
   - 自动识别：数据、核心概念、强调词汇

3. **智能标记规则**
   - 每句话必须有情绪标记
   - 问句自动添加[疑问地]
   - 重要数据和概念自动标记【】

#### TTS层优化

**文件**：`utils/tts_generator.py`

**新增功能**：
1. 在文本中嵌入演绎指导
2. 说明情绪标记含义
3. 指导重音和停顿处理

3. TTS请求时间更长，代理可能超时

### 解决方案

#### 方案1：等待代理稳定后重试（推荐）

```bash

# 生成v1.2 TTS音频
python main.py tts output/translations/v1.2_optimized_20251215_231954.json \
  -o output/audio_segments/v1.2_final \
  -m output/audio_segments/manifest_v1.2_final.json

# 合并音频
python main.py merge output/audio_segments/manifest_v1.2_final.json \
  -o output/final/podcast_v1.2.mp3
```

#### 方案2：使用完整pipeline（自动重试）

